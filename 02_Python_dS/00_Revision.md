
# Day 2 Quick Revision Sheet

## Strings
- Ordered sequence of characters
- Immutable
- Common methods: lower(), upper(), strip(), replace(), split()
- Slicing: text[start:end:step]

## Lists
- Ordered and mutable
- append(), insert(), remove(), pop(), sort()
- List comprehensions:
```python
squares = [x*x for x in range(10)]
```

## Tuples
- Ordered and immutable
- Faster than lists
- Useful for fixed data

## Sets
- Unordered collection of unique values
- union(|), intersection(&), difference(-)
- Great for removing duplicates

## Dictionaries
- Key-value pairs
- Fast lookups
- keys(), values(), items()

## AI Engineer Essentials
- Iterate through lists and dictionaries comfortably
- Understand mutability
- Use comprehensions
- Know when to use list vs tuple vs set vs dictionary

## Interview Questions

3. Difference between append() and extend()?
4. What is dictionary comprehension?
