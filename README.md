## Setup

Create a virtual environment:
```bash
python -m venv venv
```

Activate it:
```bash
# Windows
venv\Scripts\activate

# macOS/Linux
source venv/bin/activate
```

Install dependencies:
```bash
pip install -r requirements.txt
```

To run notebooks:
```bash
pip install jupyter
jupyter notebook
```
